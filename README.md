# Pokemon Wallet

Pokemon Wallet adalah aplikasi dompet digital sederhana berbasis Flutter yang terintegrasi dengan Pokemon Marketplace. Aplikasi ini memungkinkan pengguna untuk login menggunakan akun yang sama dengan Marketplace, melakukan top up saldo, dan melihat riwayat transaksi yang telah dilakukan.

## Fitur

- Login menggunakan akun Pokemon Marketplace
- Menampilkan saldo pengguna
- Top Up saldo wallet
- Menampilkan riwayat transaksi
- Menampilkan transaksi terbaru
- Integrasi dengan Firebase
